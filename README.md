[![Firebase](https://img.shields.io/badge/Firebase-Auth%20%26%20Sync-orange.svg)](https://firebase.google.com/)
[![PWA](https://img.shields.io/badge/PWA-Offline--First-blue.svg)](https://developer.mozilla.org/en-US/docs/Web/Progressive_web_apps)

**Barakah Planner** — это высокотехнологичная экосистема продуктивности для мусульман, построенная по философии **SurviveKit**. Мы объединили духовные практики и современные методы планирования в одном защищенном, оффлайн-первом приложении.

ИншаАллах, это поможет вам достичь бараката в делах и жизни.

---

## 🚀 Основные возможности

### 🕋 Духовный фундамент

- **Ибадат-трекер:** Отслеживание намазов (с визуальной пульсацией при 100% выполнении), азкаров и чтения Корана.
- **Хиджра Календарь:** Автоматический расчет дат и подсветка **Сунна-постов** (понедельник и четверг).
- **Ният-модуль:** Начинайте каждый день с осознанного намерения.

### 🧠 Планирование и Творчество

- **Интеллектуальный Планер:** Расписание, привязанное к временам намаза (Фаджр-Зухр и т.д.).
- **Tafakkur Board:** Бесконечный холст (Infinity Canvas) для рисования, заметок и майндмэппинга. Работает быстро и плавно даже на мобильных.
- **AI Assistant (Gemini):** Личный помощник, который поможет составить расписание на день (требуется ваш API-ключ).

### ☁️ Синхронизация и Безопасность (NEW)

- **Firebase Sync:** Автоматическая синхронизация данных между вашими устройствами.
- **Multi-Auth:** Вход через Google или Email.
- **SurviveKit Backup:** Экспорт всех данных в один JSON-файл за секунду. Ваши данные принадлежат только вам.

## 🛠 Техническая информация

Приложение спроектировано как **Offline-First PWA**. Оно работает в самолете, в горах и вообще без интернета.

- **Stack:** Vanilla JS (ES Modules), Tailwind CSS v4, Firebase (Auth/Firestore).
- **Storage:** LocalStorage + IndexedDB (для тяжелых рисунков).
- **Zero Build:** Максимальная производительность без лишних фреймворков.

## 📦 Установка и запуск

1. Склонируйте репозиторий.
2. Откройте `index.html` в любом современном браузере.
3. Или установите как PWA через иконку в адресной строке.

**Для разработчиков (Firebase):**
Используйте `npx firebase-tools deploy` для выкладывания на хостинг.
