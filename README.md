# 🌙 Barakah Planner (Islamic Planner & Tafakkur Board)

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)

**Barakah Planner** (Islamic Planner) — это уникальное offline-first PWA-приложение, созданное для мусульман, стремящихся к балансу между дуньей (мирским) и ахиратом (вечным).

Приложение объединяет в себе исламский трекер поклонения (Ибадат), ежедневного планировщика, постановщика глобальных целей и бесконечную доску для размышлений (Tafakkur Board), а также имеет встроенного **ИИ-Ассистента** на базе Google Gemini, который помогает составить идеальное расписание.

## ✨ Ключевые особенности

- **🌍 Глобальная локализация:** Поддержка 4 языков (Английский, Русский, Казахский, 아рабский (Arabic) с полной поддержкой RTL-верстки).
- **🕌 Ибадат-трекер:** Удобное отслеживание 5 обязательных намазов, утренних и вечерних азкаров, чтения Корана и ежедневной садаки.
- **📅 Planner & Calendar:** Интеллектуальное расписание с делением дня по временам намаза (Фаджр-Зухр, Зухр-Аср и т.д.). Маркировка дней сунна-постов (понедельник и четверг).
- **✨ AI Assistant (Gemini):** Встроенный ИИ-помощник, который поможет грамотно распределить задачи и рутину. (Требуется ваш личный Gemini API ключ).
- **♾️ Tafakkur Board:** Бесконечный холст (Infinity Canvas) для майндмэппинга, заметок и зарисовок с поддержкой мобильных жестов (Pinch-to-zoom).
- **🌙 Идеальный Dark Mode:** Приятная стилистика бумажного блокнота, которая заботится о ваших глазах ночью.
- **📴 100% Offline (PWA):** Приложение работает без интернета! Все данные надежно хранятся локально на вашем устройстве (LocalStorage + IndexedDB). Поддержка Импорта/Экспорта резервных копий.

## 🚀 Установка и запуск

Проект поставляется как единый монолитный вид (Zero Build Steps для продакшена). Вы можете просто открыть `dist/index.html` в любом современном браузере!

### Подключение ИИ

Для активации умного планировщика:

1. Запустите приложение.
2. Нажмите на фиолетовую кнопку **"ИИ Ассистент"** в правом нижнем углу.
3. Нажмите шестеренку (Настройки) и вставьте ваш бесплатный токен от Google Gemini.
4. Ключ безопасно сохранится в вашем браузере.

## 🛠 Технологический стек

- **Core:** HTML5, Vanilla JavaScript (без тяжелых фреймворков для максимальной скорости).
- **Styling:** Tailwind CSS (v4).
- **Storage:** LocalStorage (для текста), IndexedDB (для рисунков на доске).
- **AI Integration:** Google Gemini REST API.
- **Build System:** Кастомный Node.js скрипт сборки монолита (`build-monolith.js`).
