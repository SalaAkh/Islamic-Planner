# Техническая Архитектура (SurviveKit)

МашаАллах, наш код чист, прост и надежен как автомат Калашникова. Вот как все устроено под капотом.

## 🏗️ Стек Технологий

- **HTML5:** Семантическая разметка (один файл `index.html` как точка входа).
- **CSS3 / TailwindCSS:** Дизайн через классы Tailwind, кастомные анимации и переменные в `style.css`.
- **Vanilla JavaScript (ES5/ES6):** Никаких тяжелых фреймворков (React, Vue). Мы напрямую манипулируем DOM. Это дает нам 60 FPS на любых устройствах.
- **LocalStorage:** Наша главная "база данных". Абсолютно автономно.

## 📂 Структура Файлов

- `index.html` — Оболочка, верстка всех вкладок (День, Календарь, Цели, Доска).
- `style.css` — Кастомные стили (премиальные анимации чекбоксов, сетка для доски, Sticky Notes).
- `store.js` — Слой данных (Data Layer). Все методы для работы с localStorage лежат здесь.
- `board.js` — Сложный движок бесконечного холста (Zoom, Pan, Drag & Drop).
- `main.js` — Контроллер (Controller). Управляет переключением вкладок, рендером планера и календаря.

## ⚡ Оффлайн-First (Local Protocol)

Мы вырезали все ES Modules (`import`/`export`), чтобы проект работал через `file:///`.
**Зачем?** Чтобы пользователь мог скинуть папку с проектом на флешку, открыть `index.html` и начать планировать свою жизнь даже без установки локального сервера (Node.js). Максимальная отказоустойчивость.

## 💾 Управление Состоянием (State Management)

Все данные сохраняются при любом изменении (ввод текста, клик по чекбоксу).

- Ключ `barakah_day_YYYY-MM-DD` — для дневных планов.
- Ключ `barakah_goals` — для глобальных целей.
- Ключ `barakah_board_state` — для сохранения стикеров на доске (координаты X, Y, текст).
