# Функционал (Features)

Детальное описание модулей Barakah Planner.

## 1. Дневной Планер (Daily View)

Модуль для ежедневного контроля.

- **Ният (Намерение):** Окошко для напоминания себе, ради чего делается сегодняшняя работа.
- **Ибадат Трекер:** Чекбоксы для намазов (Фаджр, Зухр, и т.д.) и азкаров.
- **Прогресс-бар:** При 100% заполнении Ибадат Трекера включается золотая пульсация (премиальный микро-интеракшен).
- **Временные блоки:** Разделение дня на исламские интервалы (От Фаджра до Зухра, от Зухра до Асра).
- **Динамические задачи:** В каждый временной блок можно добавить произвольное количество задач. Задачи сохраняются в LocalStorage с привязкой к блоку и восстанавливаются при перезагрузке (фикс TypeError с CSS-селектором `.block-id`).
- **Увеличенные отступы:** Между задачами внутри блоков добавлены увеличенные вертикальные отступы для удобства чтения (принудительный инлайн-стиль, устойчивый к Tailwind-конфликтам).
- **Textarea для ввода:** Поля ввода сделаны через `<textarea>` с авто-ресайзом (через `main.js`). Текст переносится на новую строку, не обрезается.
- **AI Planner (Gemini):** Умный ассистент, который генерирует план дня на основе ваших пожеланий, учитывая время намазов и приоритеты.
- **Локализация 4.0 (Seamless URL Sync):** Полная поддержка RU, KK, AR, EN с автоматическим переключением направления текста (RTL для арабского). Переключение языка происходит мгновенно без раздражающих перезагрузок страницы благодаря HTML5 History API (`?lang=`), генерируя красивые делящиеся ссылки.

## 2. Tafakkur Board (Бесконечная Доска)

Наша киллер-фича. Пространство для Мозгового Штурма и Рисования (Miro/Notion-style).

- **Рисование:** Встроенный инструмент рисования с поддержкой выбора толщины, цвета и ластика (идеально для планшетов и Apple Pencil).
- **IndexedDB Storage:** Рисунки сохраняются в продвинутую базу данных браузера (IndexedDB), что снимает ограничения по объему хранимых данных.
- **Масштабирование (Zoom) и Панорамирование (Pan):** Навигация как в картах. Зажмите ПРОБЕЛ, чтобы передвигать доску. На мобильном работает через touch.
- **Стикеры (Sticky Notes) — 4 цвета:** Добавляем записки 4-х цветов:
  - 🟡 **Желтые** — классика, для заметок.
  - 🟢 **Зеленые** — для задач и планов.
  - 🔵 **Синие** — для идей и концепций.
  - 🩷 **Розовые** — для вдохновения и мечтаний.
- **Защита контента:** Рисунки и изображения на доске защищены от скачивания через `contextmenu` и `dragstart` блокировщики.
- **Свободное перемещение:** Развешивайте идеи визуально, стройте карты мыслей (Mind Maps).

## 3. Исламский Календарь (Календарь Хиджры)

- Рядом с Григорианской датой автоматически отображается месяц и день по Хиджре (используется нативный `Intl.DateTimeFormat`).
- **Сунна-посты:** Понедельники и Четверги автоматически подсвечиваются золотым цветом с бейджем "Сунна".

## 4. Глобальные Цели (Ахират и Дунья)

Система планирования, делящая цели на две вечные категории:

- Приготовления к вечной жизни (Коран, Благотворительность, Исправление характера).
- Мирские дела (Бизнес, Здоровье, Семья).

## 5. SurviveKit: Резервное копирование и PWA

Мы гарантируем выживание ваших данных в любых условиях.

- **Умный Экспорт/Импорт (JSON):** В левом нижнем углу доступны кнопки бэкапа. Одно нажатие — и все ваши задачи, цели, стикеры и даже рисунки с доски скачиваются в одном файле.
- **100% Оффлайн (PWA):** Приложение можно "Установить" на экран смартфона или ПК. Оно кэширует шрифты и иконки. Отключите интернет — Barakah Planner даже не заметит разницы.

## 6. Локальный Режим (Local Mode)

- **Автозапуск без интернета:** При открытии через `file://` приложение обходит Firebase авторизацию автоматически.
- **Локальный пароль:** Показывается кастомный промпт для ввода локального пароля вместо Firebase Auth окна.
- **Индикатор:** Бейдж "🏠 Local Mode" в верхней части интерфейса информирует пользователя о текущем режиме.
- **Безопасность данных:** Данные автономно хранятся в LocalStorage/IndexedDB устройства без отправки на сервер.

## 7. Поддержка Проекта (Donation System)

Система для сбора средств на развитие и баракат проекта.

- **Donation Toast:** Приятное всплывающее уведомление-сердце, появляющееся каждые 3 минуты (на 15 секунд), чтобы не мешать работе.
- **Kaspi Support:** Прямая интеграция переводов Kaspi (Казахстан) с кнопкой быстрого копирования номера.
- **Global Support:** Ссылки на Patreon и Boosty для пользователей со всего мира.

*Все изменения подкрепляются всплывающим UX-тостом (МашаАллах, сохранено!), что создает чувство надежности.*

## 8. Продвинутая Безопасность (Security & Anti-Abuse)

Приложение спроектировано с учетом современных угроз:

- **Защита Базы Данных:** Интегрированные правила Firebase Firestore блокируют любые попытки записи несанкционированных данных, защищая бюджет проекта от мусорного трафика.
- **HTTP-броня:** Статические файлы отдаются с жесткими заголовками безопасности (Anti-Clickjacking, XSS Protection, HSTS), что гарантирует безопасность данных пользователя даже в публичных Wi-Fi сетях.
- **Защита изображений:** Контентные фото защищены от скачивания, копирования и открытия в новой вкладке через комбинацию CSS и JS-перехватчиков событий.
