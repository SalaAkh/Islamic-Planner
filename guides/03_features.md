# Функционал (Features)

Детальное описание модулей **Islamic Planner (Barakah Planner)**.

## 1. Landing Page (Привлечение)

- 4-х языковый лендинг (RU, KK, AR, EN) с переключением на лету и автоопределением по URL (например `islamic-planer.web.app/kk`).
- Визуально-премиальный дизайн: glassmorphism, градиенты, анимации параллакса, демо-ролик (GIF).
- Кнопка "Начать бесплатно" перекидывает пользователя прямо в `app.html?lang={current}`.

## 2. Дневной Планер (Daily View)

Модуль для ежедневного контроля.

- **Ният (Намерение):** Окошко для напоминания себе, ради чего делается сегодняшняя работа.
- **Ибадат Трекер:** Чекбоксы для намазов (Фаджр, Зухр, Аср, Магриб, Иша).
- **Временные блоки:** Разделение дня на исламские интервалы (От Фаджра до Зухра и т.д.).
- **Умное поле Дата/Хиджра:** Рядом с Григорианской датой отображается Исламский месяц и год.
- **AI Planner (Gemini):** Умный ассистент, генерирующий план дня на основе пожеланий пользователя (Инкогнито ключ либо ключ пользователя).
- **UX Фишки:** Авторасширяющиеся текстовые поля (`textarea` resize) без полос прокрутки.

## 3. Исламский Календарь (Календарь Хиджры & Event System)

- Полновесный визуальный календарь-сетка.
- **События (Events/Reminders):** Создание событий 5 типов (Исламские, Личные, Работа, Семья, Здоровье).
- **Напоминания (уведомления):** Работают через браузерный API (Web Push API - Local browser notifications), бесплатно и оффлайн. С сохранением в Firebase `reminders`.
- **Сунна-посты:** Подсказки на дни Сунны (Понедельник и Четверг).

## 4. Tafakkur Board (Бесконечная Доска)

Наше пространство для свободного размышления (Miro/Notion-style).

- **Рисование:** Поддержка мыши и Apple Pencil (Толщина, цвет, ластик).
- **Навигация:** Масштабирование (Zoom - Scroll/Pinch) и панорамирование (Pan - пробел + мышь / Pan tool).
- **Стикеры (Sticky Notes) — 4 цвета:** Желтые (заметки), Зеленые (задачи), Синие (идеи), Розовые (мечты).
- Инлайн редактирование текста на стикерах.

## 5. Глобальные Цели (Ахират и Дунья)

Разделение планирования на:

- **Ахират (Вечность):** Изучение Религии, Пожертвования (Садака Джария), Нравственность.
- **Дунья (Мирское):** Дела бизнеса, семья, здоровье, обучение.

## 6. Activity Log (Аудит действий)

- Журнал действий: "вошел в систему", "добавил стикер", "сохранил 10 задач" и т.д.
- Мотивирует (Streak system). Хранится и локально, и синхронизируется с облаком (коллекция `users/{uid}/activity_log`).

## 7. Выживание: Резервное копирование и PWA

- **100% Оффлайн:** PWA ставится как нативное приложение (iOS/Android/Windows).
- **Import / Export Backup:** Выгрузка всей базы (доска, задачи, логи) в один JSON. Если кэш сотрётся, жизнь можно будет восстановить.

## 8. Монетизация и Поддержка Проекта

- Кнопка "Помочь проекту" и Kaspi-интеграция.
- Лёгкие всплывающие уведомления, предлагающие инвестировать в баракатный проект.
